Projects

RHAZ: Last Man Space Survival Ep2

Rhaz is a 2D game created using Unity Game Engine for the purpose of participationg in Global Game Jam 2020. This game is centered around the theme of "Repair". Rhaz takes place in a distant future where all humanity had been extinguished and a man has accidentally crashed his ship on an unknown planet. He is ambushed with monsters and needs to collect their orbs to repair his ship.
Technologies used: C#, Unity Game Engine

GeekText

GeekText is a web-based bookstore aiming towards a particular niche in technology to allow them a place to buy books online. I was assigned the feature of profile management, which allowed user functionality to create and maintain persistent data in a database. The site allows users to manage their login credentials, personal information, credit card information, and shipping address.
Technologies used: HTML5, CSS3, Javascript ES6, Firebase SDK
